Built an Android application for organizing basic pet care information, including pets, veterinary appointments, vaccines, treatments, and local reminders.

The app includes local email-based login, pet creation and editing, per-pet detail screens, upcoming appointments and treatments, deletion with confirmation and undo, and a settings screen with the current user session.

The project is developed in Kotlin using Android SDK, AndroidX, Material Components, ViewBinding, Room, Coroutines, Flow, RecyclerView, Coil, and Gradle Kotlin DSL.

MyPetManager is designed as a local-first app that does not depend on external services or private API keys.
