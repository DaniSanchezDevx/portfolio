Built a professional portfolio as the public delivery project for the Evolve Master in Data Science, bringing together projects, skills, education, experience, and contact workflows in one polished site.

The site uses TanStack Start with file-based routing, Tailwind CSS, lucide-react icons, and Content Collections so projects, education, jobs, and blog content can be managed through typed Markdown files.

The contact flow runs through a Vercel server route and Resend, keeping API credentials server-side while allowing visitors to send messages directly from the site.

The project is deployed on Vercel and structured so new portfolio entries can be added by dropping Markdown files into the content directory, making it easy to maintain as a living technical showcase.
