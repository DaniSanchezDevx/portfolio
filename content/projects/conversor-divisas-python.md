Built a Python application for converting currencies with up-to-date exchange rates and multiple ways to use the tool depending on the scenario.

The project supports one-off terminal conversions, a graphical interface with Tkinter, CSV-based batch conversions, and currency listing utilities.

It uses the Frankfurter public exchange-rate API and does not require third-party packages or private API keys for its core functionality.
