Built a recipe generation app with Streamlit and OpenAI that creates personalized dishes from the ingredients the user already has at home.

The project includes a guided form for ingredients, servings, cooking style, dietary preferences, and difficulty, then returns a unique recipe rendered inside the app.

It also supports PDF export, environment-based configuration, and direct API key input from the sidebar for local testing.
